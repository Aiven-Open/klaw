Architecture

Front End

Front end is built with AngularJs, Bootstrap 4, HTML, CSS, and Java script. FE interacts with User interface Rest Api.

Frameworks

Spring boot, Spring Security, Spring Data, Hibernate, Kafka AdminClient Api, Datastax, Angular Js, HTML, CSS, Java script

User Interface Api

Built with Spring boot, spring security frameworks, interacts with Front end, Cluster API and Metastore. Plays a major role in Kafkawize.

Metastore

Apache Cassandra or RDBMS datastore stores all users, teams, topicRequests, request and execution data from all the users..

Cluster Api

Built with Spring boot, spring security frameworks, interacts with User Interface Api. Kafka AdminClient Api to interact with Clusters.

Roles & Authorizations

Users with USER role can request for components and users with ADMIN role can approve. SUPERUSER role have full access including Synchronizing Metadata.
